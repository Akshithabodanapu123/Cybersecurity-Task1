# Linux Notes

## Introduction

Linux is an open-source operating system widely used in cybersecurity and ethical hacking.

## Basic Linux Commands

### pwd

Displays the current working directory.

### ls

Lists files and folders in the current directory.

### cd

Changes the current directory.

### mkdir

Creates a new folder.

### touch

Creates a new file.

### chmod

Changes file permissions.

### chown

Changes file ownership.

### apt

Package manager used to install and update software.

### ifconfig

Displays network configuration information.

### ping

Checks network connectivity.

### netstat

Displays active network connections.

## Conclusion

Linux is an essential operating system for cybersecurity professionals because it provides powerful tools for networking, security testing, and system administration.
